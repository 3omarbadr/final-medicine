# Health Awareness

## Installation

Please follow these steps tp start the app Run.

```
cp .env.example .env
```

```
composer install
```

```
php artisan key:generate
```

```
npm install
npm run dev
npm run build
```

```
php artisan migrate
php artisan db:seed
```

```
php artisan storage:link
```

```
php artisan serve
```
